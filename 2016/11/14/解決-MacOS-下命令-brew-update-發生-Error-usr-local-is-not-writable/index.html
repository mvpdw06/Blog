2016-11-14
#Mac, MacOS, Yarn, Brew, Brew-update, Error

起源

會有這篇是因為我想要在 MacOS 上面安裝 Yarn 套件管理器，而 Yarn 需要比較新版的 brew 才有辦法安裝，所以我們需要先 update 我們的 brew，才能夠完成安裝。而這是我在 update brew 的時候遇到的問題，解法如下：

我的 MacOS 環境

  • MacOS: 10.12.1
  • Terminal: iTerm2
  • Bash-Shell: Fish-Shell
